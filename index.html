Markdown Previewer

  • Syntax

    Headings

    To insert heading tags <h1></h1>,<h2></h2>,<h3></h3>,<h4></h4>,<h5></h5>,<h6></h6>, type # 1/2/3/4/5/6 times follwed by space and your text (# for <h1>,## for <h2> and so on)

    Example:

    # This is <h1></h1> heading.

    ## This is <h2></h2> heading.

    ### This is <h3></h3> heading.

    #### This is <h4></h4> heading.

    ##### This is <h5></h5> heading.

    ###### This is <h6></h6> heading.

    Text Decoration

    Enclose text in ** to make it bold, _ to italicise it, ∼∼ to strikethrough it, blockquotes by > and typing in the text, or combine these for multiple styles

    Example:

    **Bold Text**

     gives Bold Text

    _Italic Text_ gives Italic Text

    ∼∼Strikethrough∼∼ gives Strikethrough

    Links

    Enclose link text in [] followed by complete URL in ()

    Example:

    [Google](https://www.google.com/) gives Google

    Tables

    You can add tables using the syntax below (don't forget to add an empty line before and after the table markdown)

    Column Header | Column Header | Column Header
------------- | ------------- | -------------
Data/Value    | Data/Value    | Data/Value
Data/Value    | Data/Value    | Data/Value

    Example Output:

    Column HeaderColumn HeaderColumn Header
    Data/ValueData/ValueData/Value
    Data/ValueData/ValueData/Value

    Unordered lists

    Type ‐ follwed by text to make an unordered bulleted list. You can indent the list by adding extra spaces before the ‐.

    Example Input:

    - List Item
  - List Item
     - List Item
        - List Item

    Example Output:

    • List Item
      • List Item
        • List Item
          • List Item

    Ordered Lists

    Type text preceeded by 1. or 2. and so on to make a numbered list.

    Example Input:

    1. List Item
2. List Item
3. List Item

    Example Output:

    1. List Item
    2. List Item
    3. List Item

    Images

    To insert an image, type ![alternate](URL-of-image) and replace alternate by the alternate text you want to display when image is unavailable, and URL-of-image by the URL address of the image you want to insert.

    ![Google Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/2/2f/Google_2015_logo.svg/368px-Google_2015_logo.svg.png)
